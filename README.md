# Stellar Soroban Insurance Claim System





## Project Description

The **Stellar Soroban Insurance Claim System** is a decentralized smart contract built using the **Soroban smart contract platform on the Stellar blockchain**.

This project demonstrates how blockchain technology can be used to create a **transparent, secure, and tamper-proof insurance claim management system**.

Traditional insurance claim systems rely on centralized authorities which can lead to delays, lack of transparency, and potential fraud. By leveraging **Stellar's Soroban smart contracts**, this project ensures that claim records are stored on-chain and cannot be modified without authorization.

This project was built as a learning project to explore **Web3 development and Soroban smart contracts using Rust**.

---

## What It Does

The smart contract allows users to interact with a decentralized insurance claim system.

Users can:
- Submit insurance claims
- Store claim data securely on the blockchain
- Track claim status

Administrators can:
- Approve insurance claims
- Update claim status

All claim records are stored on the **Stellar blockchain**, making the system transparent and verifiable.

---

## Features

• Submit new insurance claims  
• Store claim details on blockchain  
• Approve claims by authorized entity  
• View claim information anytime  
• Immutable claim records  
• Built using **Rust + Soroban SDK**

## Smart Contract Functions

| Function | Description |
|--------|-------------|
| `submit_claim()` | Submit a new insurance claim |
| `approve_claim()` | Approve an existing claim |
| `view_claim()` | Retrieve claim details |
